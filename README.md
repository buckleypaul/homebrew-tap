# Homebrew Tap

This repository contains Homebrew formulae for my personal projects.

## Installation

```bash
brew tap buckleypaul/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| blescan | Terminal UI application for scanning and analyzing BLE device advertisements |
| haide | AI File Exclusion Manager for Git |

## Usage

After tapping, install any formula:

```bash
brew install blescan
brew install haide
```
